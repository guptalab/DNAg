<p> <b><i>Abstract</i></b> <br>


	<div id="wrapper">
		<p> This project is about designing a suitable software that can be used as a tool for teaching purpose as well as to generate DNA Self Assembly structures and DNA sequences which could be further used for experimental purposes. DNA has extremely dense information storage and parallel computing power, this makes it a unique computational element to study.
  	 The motivation to take up this topic came through the courses of ‘Natural Computing’, ‘Introduction to Coding Theory’ and ‘Models of Computation’. The topics such as  - ‘DNA Self Assembly’, ‘DNA Computation’, worked as an interlink between the three structure and developed an interest to continue with this project. Since a single Software to learn all the  concept was a something which would have helped us a lot, so we took is as an initiative to prepare one ourselves.
This objective was to address both the issues of generating DNA Self Assembly using DNA
Computation and creating a platform which answers to the doubts of a layman about this topic and understanding of the various concepts that lead to the results. The tool was aimed to
extend a user friendly interface and at the same time provide with reliable results which could be used for experimental purposes. 
The DNA Self Assembly Generator generates most stable structures of the DNA Self Assembly formed by particular set of grammar rules.  Self Assembly is a process in which components obtain the most stable structure possible. DNA Self Assembly we try to illustrate the interaction between strand at nano level to generate products which are stable in equilibrium and have minimum free energy. The tool DNAg- ’The DNA Self Assembly Generator’, begins its flow of working by checking the Grammar Rules satisfying the Languages (Context Free Language and Regular Language) and designing DNA code words to generate thermodynamic-ally stable DNA sequences that are then used to generate this computational model through the DNA Self Assembly.  Based on some simple concept of “Models of computation” we checked the input language and grammar. Further, to represent the Language we require stable DNA code words. So, our second task was to construct the stable Single strand DNA structure (sequence of nucleotides {A,C,G,T}. To do so our focus was to generate large set of DNA code words with certain combinatorial constraints like GC constraint, Hamming Distance, Reverse Hamming Distance and Thermodynamic stability. Those sets of code words are expected to give a stable structure thus not including any secondary structure formation.
             The software works for regular grammar and could be extended for context free and context sensitive grammars. The codebook is generated for word lengths from 1 to 80 and could be extended further. 
     This project was done under the guidance of Prof. Manish.K.Gupta.As a done in partial fulfillment of the requirements towards the award of Btech. (ICT) Degree -2013
</p>
